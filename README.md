# ⌚ Chronos Luxury Boutique - מדריך הפעלה

ברוך הבא לחנות השעונים החדשה שלך! האתר בנוי בטכנולוגיות המתקדמות ביותר (React + Gemini AI) ומוכן למכירות.

## 🚀 איך להעלות את האתר לאוויר (שלב אחר שלב)

### 1. GitHub - אחסון הקוד
*   כנס ל-GitHub וצור Repository חדש.
*   **שדה Description:** אתה יכול להשאיר אותו **ריק**. הוא לא משפיע על האתר.
*   **העלאת קבצים:** גרור את כל הקבצים לתיבה הכחולה.
*   **שדה Commit Message:** לפני הלחיצה על "Commit changes", גת'אב מבקש כותרת. כתוב פשוט "Initial upload" (זה חובה, התיאור שמתחתיו הוא רשות).

### 2. Vercel - הפיכה לאתר חי
*   חבר את חשבון ה-GitHub ל-Vercel.
*   בחר ב-Project שהעלית ולחץ על **Import**.
*   **הגדרת המפתח (קריטי):** תחת "Environment Variables", הוסף Key בשם `API_KEY` עם הקוד שקיבלת מ-Google AI Studio.

### 3. דומיין (כתובת האתר)
*   Vercel ייתן לך כתובת זמנית. תוכל לשנות אותה בהגדרות ה-Project תחת "Domains".

## 💰 ניהול החנות (בקובץ constants.ts)

*   **מחירים:** עדכן את ה-`price` לכל שעון.
*   **תמונות:** תוכל להחליף את הקישור ב-`image` לכל תמונה שתבחר.
*   **תשלום:** בתוך `STORE_CONFIG`, הדבק את לינק התשלום שלך ב-`checkoutLink`.

---
### 💡 טיפ חשוב על ה"Description" שראית בקוד:
ה-`Description` שמופיע בתוך ה-XML שאני שולח לך נועד רק להסבר שלי אליך. בתוך האפליקציה עצמה, התיאור היחיד שחשוב הוא ב-`metadata.json` (זה מה שגוגל יציג כתיאור האתר).
